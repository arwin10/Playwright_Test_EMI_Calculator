üêõ TROUBLESHOOTING
================================================================================

ISSUE: npm install fails
FIX: Delete node_modules and package-lock.json, run npm install again

ISSUE: Browsers not installed
FIX: Run: npx playwright install --with-deps

ISSUE: Tests timeout
FIX: Increase timeout in .env file (TIMEOUT=60000)

ISSUE: Port conflict
FIX: Check if port 3000 is available

ISSUE: Permission denied
FIX: Run with sudo (macOS/Linux) or as Administrator (Windows)

ISSUE: Module not found
FIX: Ensure you're in the correct directory and ran npm install
